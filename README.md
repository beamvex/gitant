# GitHub (gh) Explorer (VS Code Extension)

This extension adds an Explorer-like Tree View that shows your GitHub account using the `gh` CLI in the background.

## Requirements

- `gh` installed
- Authenticated: `gh auth login`

## Run locally

- `npm install`
- `npm run compile`
- Press `F5` in VS Code to launch an Extension Development Host.

## Features

- Explorer view: **GitHub (gh)**
- Shows: account -> repositories -> branches
- Commands:
  - `GitHub (gh): Refresh`
  - `GitHub (gh): Open in Browser` (on repo)
  - `GitHub (gh): Copy Clone URL` (on repo)
